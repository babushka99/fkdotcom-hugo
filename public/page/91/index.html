Currency Transaction Report (CTR)

Definition of Currency Transaction Report (CTR) A Currency Transaction Report (CTR) is a document that financial institutions in many countries must file with their respective financial authorities, such as the Financial Crimes Enforcement Network (FinCEN) in the United States. It is required for any transaction or series of related transactions that involve a total amount of more than a specified threshold (e.g., $10,000 in the U.S.) in cash.
