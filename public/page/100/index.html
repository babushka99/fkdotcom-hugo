HS Code

HS Code, short for Harmonized System Code, is a standardized numerical method of classifying traded products. It is used by customs authorities around the world to identify products when assessing duties and taxes and for gathering statistics. The HS Code system is maintained by the World Customs Organization (WCO), an independent intergovernmental organization with over 180 member countries based in Brussels, Belgium. Definition and Purpose The HS Code is a six-digit code that classifies and defines internationally traded goods. It’s based on the Harmonized Commodity Description and Coding System.
