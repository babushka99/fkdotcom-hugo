Average Transaction Value (ATV)

What is Average Transaction Value (ATV)? Average Transaction Value (ATV) is a key performance metric used in retail, e-commerce, and other industries to measure the average amount of money a customer spends per transaction. It is calculated by dividing the total revenue generated over a specific period by the total number of transactions during that same period.
